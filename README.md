# Digital Trade

A premium, high-performance crypto trading and investment application.

## Features
- **Standalone Demo Mode**: Fully functional offline demo using `localStorage`.
- **Real-time Profit Simulation**: Watch your profits grow live on the dashboard.
- **Admin Power Panel**: Manage users, balances, and global settings.
- **Responsive Design**: Beautiful, glassmorphic UI optimized for all devices.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Stonerdeveloper/digitalTrade.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Demo Access
On the login screen, use the one-click demo buttons:
- **Demo User**: `user@demo.com`
- **Demo Admin**: `admin@demo.com`

No registration or database setup required for demo sessions!
